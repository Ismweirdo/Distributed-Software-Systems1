## 一、项目概述

本项目是一个**分布式即时通讯系统**，核心特色是在传统 IM 功能之上集成 **AI Bot 化身系统**——用户可导入聊天记录自动生成具有特定人物语言风格的 AI 机器人，支持多级记忆（短期/长期/RAG 检索）和流式对话。

**技术栈：** Vue 3 + Spring Boot 3 + MySQL + Redis + RabbitMQ + WebSocket STOMP + Docker Compose
